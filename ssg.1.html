NAME

ssg — static site generator

SYNOPSIS

ssg [-bd]

DESCRIPTION

A tiny (~50 lines) script for creating websites with pandoc.

The program paths can be set by the following sequence:

  1. Using the per-project configuration file, ./.ssgrc
  2. Using the per-user configuration file, ~/.ssg

Options

-b, --build

Builds site

-d, --dev

Builds site, starts http server (httpwatcher) and watch daemon (entr)

[default]

Prints message.

Paths

srcdir

(default) $PWD/src

Path to input directory. Can contain files of valid pandoc input formats.

https://pandoc.org/MANUAL.html#general-options

destdir

(default) $PWD/dist

Path to output directory.

staticdir

(default) $PWD/static

Path to static directory. This directory's contents will be copied to the destdir.

template

(default) $PWD/template.html

Path to pandoc --template html file.

https://pandoc.org/MANUAL.html#templates

DEPENDENCIES

  • pandoc— https://pandoc.org/MANUAL.html
  • entr — https://eradman.com/entrproject/
  • httpwatcher — https://pypi.org/project/httpwatcher/
